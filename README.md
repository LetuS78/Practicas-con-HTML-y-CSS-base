# Practicas-con-HTML-y-CSS-base
Repositorio de práctica con HTML y CSS base. Incluye estructura semántica y estilos iniciales para proyectos web simples.



## Características
- Barra de navegación con menú desplegable.
- Logo centrado en la navbar.
- Estilos iniciales con **CSS puro** (flexbox, sombras, hover).
- Estructura semántica en HTML5.
- Diseño simple y adaptable como punto de partida para proyectos más grandes.
